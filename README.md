ModSettings

To Start add `ModSettings` to the dependencies list in the Mod creator.
Then download the Mod into the Documents/TheBus/ModsEditor folder or add the URL into the clone mod section of the modcreator.
Restart to activate the dependency.

Navigate to Content/Gameplay/ActorAttachments. Make an Advaced copy of BP_ExampleModSettings. FindReplace "Example" with your Mod Name.
